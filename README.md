# \# Stock Control System

##### 

## Inventory management system with QR code scanning, multi-user support, and interactive reports.

##### 

##### \## Features

##### 

##### \- 📦 Inventory Management - Add, edit, delete items

##### \- 📱 QR Code Scanning - Generate and scan codes to deplete stock

##### \- 📊 Interactive Reports - Clickable cards, reorder worksheets

##### \- 👥 Multi-User System - Admin and general user roles

##### \- 💰 Cost Tracking - Track unit costs and inventory value

##### \- 📥 Import/Export - CSV bulk operations

##### 

##### \## Installation

##### ```bash

##### \# Clone the repository

##### git clone https://github.com/YOUR-USERNAME/stock-control-app.git

##### 

##### \# Navigate to folder

##### cd stock-control-app

##### 

##### \# Install dependencies

##### npm install

##### 

##### \# Create environment file

##### cp .env.example .env

##### 

##### \# Start server

##### npm start

##### ```

##### 

##### \## Usage

##### 

##### 1\. Open http://localhost:3001/admin.html

##### 2\. Register first user (becomes admin)

##### 3\. Add inventory items

##### 4\. Generate QR codes

##### 5\. Use reports for reordering

##### 

##### \## Tech Stack

##### 

##### \- \*\*Backend:\*\* Node.js, Express

##### \- \*\*Auth:\*\* JWT, bcrypt

##### \- \*\*Storage:\*\* JSON files

##### \- \*\*Frontend:\*\* Vanilla HTML/CSS/JavaScript
