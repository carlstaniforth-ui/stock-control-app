🚀 Getting Started

Welcome to Stock Control!

This system helps you manage inventory, track stock levels, generate QR codes, and create reorder lists automatically.

💡 First Time User?
If you're new to the system, start by exploring the Dashboard to see your inventory, then try scanning a QR code to deplete stock.

What You Can Do

📦 Manage Inventory

Add, edit, and delete items. Track stock quantities and costs.

📱 QR Code Scanning

Generate QR codes for items. Scan to instantly deplete stock.

📊 Interactive Reports

View insights, create reorder lists, and export data.

👥 User Management

Admins can create users with different permission levels.

📦 Dashboard Guide

Adding Items

Admins can add new inventory items:

  1. Fill in Item Details
    Enter SKU, Name, Initial Stock, Location, and Description.
  2. Add Cost Information (Optional)
    Enter Unit Cost and select Currency (GBP/USD/EUR).
  3. Click "Add Item"
    Item appears in your inventory list.

Managing Items

Each item card shows:

  • Name & SKU - Item identification
  • Stock Level - Current quantity (color-coded)
  • Cost - Unit cost if entered
  • Location - Where item is stored

Available Actions:

📱 View QR - Display and print QR code for scanning
➕ Replenish - Admin Only Add stock when receiving deliveries
🗑️ Delete - Admin Only Remove item from inventory

Bulk Import/Export

Admin Only

Export CSV:

  • Click "📥 Export CSV" button
  • Downloads current inventory as CSV file
  • Use for backups or analysis in Excel

Import CSV:

  • Click "📤 Import CSV" button
  • Select CSV file with format: SKU, Name, Stock Quantity, Unit Cost, Currency, Location, Description
  • Existing SKUs will be updated, new SKUs will be created
⚠️ Import Warning: The import process updates existing items. Always export a backup before importing new data.

📊 Interactive Reports

Overview Statistics

The reports page shows four key metrics:

  • Total Inventory Value - Sum of all stock × unit costs
  • Total Items - Number of different products
  • Low Stock Items - Items with ≤5 units (clickable!)
  • Out of Stock - Items with 0 units (clickable!)

Reorder Workflow (The Best Feature! 🎯)

  1. Click "Low Stock" or "Out of Stock" Card
    Shows filtered list of items needing reorder
  2. Review Suggested Quantities
    System automatically suggests reorder amounts based on stock level
  3. Adjust as Needed
    Edit quantities or uncheck items you don't want to order
  4. Click "📤 Export Selected"
    Downloads CSV with your reorder list
  5. Email to Supplier
    Send the CSV file - it's ready to use!
⚡ Time Savings: This process takes 30 seconds instead of 20-30 minutes of manual work!

Export Formats

Three specialized formats available:

Format Best For Contains
Reorder List Supplier orders SKU, Name, Current Stock, Suggested Qty, Unit Cost, Total
Full Report Analysis & reports All item details including location, description, values
Stock Count Sheet Physical counting SKU, Name, Expected Stock, blank Actual column to fill in

Location & Value Analysis

  • Click location rows - See all items in that location
  • Top Value Items - See your most valuable inventory
  • Quick Actions - Fast filters for common views

📱 QR Code Scanning

How QR Codes Work

Each inventory item has a unique QR code that links to a mobile-friendly scan page.

Generating QR Codes

  1. Click "📱 View QR" on Any Item
    QR code appears in a popup window
  2. Print the QR Code
    Use Ctrl+P or click Print button
  3. Attach to Physical Item
    Stick label on shelf, bin, or product packaging

Scanning QR Codes

  1. Scan with Phone Camera
    Point camera at QR code (no app needed on most phones)
  2. Tap the Link
    Opens scan page showing item details
  3. Enter Quantity
    How many items are you taking?
  4. Click "✓ Confirm"
    Stock is depleted instantly with haptic feedback
📱 Mobile Friendly: The scan page is optimized for phones - large buttons, easy number input, and instant feedback.

Use Cases for QR Codes

  • Warehouse Picking - Scan when taking items from stock
  • Tool Checkout - Track who's using equipment
  • Supply Room - Easy depletion for office supplies
  • Retail Stock - Update inventory as items are sold

Creating New Users

  1. Navigate to User Management
    Click "👥 Users" button in header
  2. Fill in User Details
    Username, password (min 6 characters), and role
  3. Select Role
    Administrator (full access) or General User (view + scan only)
  4. Click "Create User"
    New user appears in the list
  5. Share Credentials
    Give username and password to the person

First User is Admin

The very first person to register becomes an administrator automatically. After that, only admins can create new users - public registration is disabled for security.

⚠️ Cannot Delete Yourself: For safety, you cannot delete your own admin account. Create another admin first if you need to remove yourself.

🔄 Common Workflows

Weekly Reorder Process

Monday Morning: Open Reports page
Click: "Low Stock: 12" card
Review: 12 items needing reorder
Adjust: Quantities if needed
Uncheck: Items for different suppliers
Export: Click "Export Selected"
Email: CSV to supplier
Done: 2 minutes total!

Receiving Stock Delivery

  1. Receive Delivery
    Physical items arrive at warehouse
  2. Open Dashboard
    Go to admin.html
  3. Find Item
    Locate item in inventory list
  4. Click "➕ Replenish"
    Admin only feature
  5. Enter Quantity
    How many units received
  6. Confirm
    Stock updated instantly

Month-End Stocktake

  1. Export Stock Count Sheets
    Reports → Export → Stock Count Sheet (by location)
  2. Print Sheets
    One per location or section
  3. Physical Count
    Staff count actual items
  4. Compare
    Expected vs Actual stock
  5. Update System
    Adjust quantities for discrepancies
  6. Investigate
    Look into large differences (shrinkage/errors)

Daily Operations

Staff Arrive: Login to scan page
Take Items: Scan QR code
Enter Quantity: How many taking
Confirm: Stock depletes automatically
Manager Checks: Reports for low stock alerts
Weekly: Review and reorder

💡 Tips & Tricks

Best Practices

  • Use Clear SKUs - Format like "WIDGET-001" or "TOOL-HAMMER-01"
  • Add Locations - Makes finding items easier (e.g., "Warehouse A, Shelf 3")
  • Enter Costs - Enables value reporting and better insights
  • Print QR Codes Clearly - Laminate for durability in warehouses
  • Regular Backups - Export CSV weekly as backup
  • Stock Checks - Monthly physical counts recommended

Keyboard Shortcuts

  • Ctrl+P - Print QR code (when viewing QR)
  • Ctrl+F5 - Refresh and clear cache
  • Tab - Navigate between form fields quickly

Mobile Tips

  • Bookmark Scan Page - Quick access on warehouse phones
  • Use Native Camera - No QR scanner app needed (iOS/Android)
  • Large Touch Targets - Scan page optimized for gloves

Reporting Tips

  • Click Don't Scroll - Use clickable cards to filter quickly
  • Export Templates - Use Stock Count Sheet for physical audits
  • Location Views - Click locations to see specific warehouse/room
  • Reorder Worksheet - Pre-checked items, just adjust and export!

🆘 Getting Help

Common Questions

Q: I forgot my password, how do I reset it?

Contact an administrator to create a new password or create a new account for you.

Q: Can I edit an existing item?

Currently, use CSV export → edit in Excel → import to update items. UI edit button coming soon!

Q: Why can't I see the "Add Item" form?

You need administrator privileges. Contact your admin for access.

Q: The QR code won't scan

Ensure good lighting, camera is in focus, and QR code is printed clearly. Most phones scan automatically with the native camera app.

Q: How do I delete multiple items at once?

Export CSV, remove rows for items to delete, then import. Or delete individually from dashboard.

Q: Can I use this offline?

No, the system requires internet connection to sync data between devices.

📧 Support: For technical issues, contact your system administrator or IT support team.
